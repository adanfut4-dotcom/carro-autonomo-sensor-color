# Carro Autónomo con Sensor de Color y OLED

## Descripción
Proyecto de control lógico en el que un carro autónomo ejecuta distintas acciones
en función del color detectado por un sensor. El sistema muestra información en una
pantalla OLED y emite señales sonoras como retroalimentación.

## Componentes utilizados
- Microcontrolador
- 4 servomotores
- Sensor de color
- Encoders
- Pantalla OLED
- Buffer
- Buzzer

## Funcionamiento
El carro detecta un color mediante el sensor y, con base en una lógica programada,
determina la acción a seguir. El color detectado y la indicación correspondiente
se muestran en la pantalla OLED y se emite un sonido cada vez que se detecta un color.

## Aprendizajes
- Implementación de lógica condicional
- Integración de múltiples periféricos
- Lectura de sensores y procesamiento de datos
- Retroalimentación visual y sonora
